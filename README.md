# Portfolio Website

👨‍💻 **About This Project**

This repository contains the code for my personal website which aims to redirect visitors to where they can learn more about my work. Designed with simplicity and user experience in mind, it's a straightforward way to demonstrate what I can do.

🔧 **What's Inside**

- `index.html`: The heart of the website, where all the magic begins.
- `styles.css` & `mobilestyle.css`: Styling for desktop and mobile views, ensuring the site looks good on any device.
- `script.js`: Adds interactivity to the web pages, making the user experience smoother and more engaging.
- `toggle.css`: Handles theme switching, because who doesn't love a dark mode?

🛠 **Skills Highlighted**

- 📐 **Web Design**: Clean, responsive design that adapts to any screen size.
- 💻 **Front-End Development**: HTML, CSS, and JavaScript used to create a dynamic user experience.
- 🌈 **CSS Mastery**: Advanced styling techniques for a polished look.
- 🔄 **Interactivity**: JavaScript magic to make the site responsive and interactive.

🚀 **Getting Started**

To get a local copy up and running, simply clone the repo and open `index.html` in your favorite browser. No complicated build steps required.
